> **작성 목적:** 이 문서는 Antigravity가 작업 의도를 완전히 이해하고 즉시 구현에 착수할 수 있도록 작성된 상세 스펙이다.

> **과외선생님 콘셉트:** 친절하고 다정하며, 교과서식 딱딱한 설명이 아닌 진짜 내 옆에서 1:1로 가르쳐 주는 과외 선생님 말투 유지

## 1. 프로젝트 개요

### 1-1. 무엇을 만드는가?

중학교 1학년 국어·영어·수학·과학·사회 5개 과목을 커버하는 **HTML5 기반 인터랙티브 스마트 학습서**다.  
매일(데일리) 특정 단원을 선택해 학습할 수 있으며, 각 단원은 아래 3단계 구조를 반드시 따른다.

```
[1단계] 개념 강의  →  [2단계] 기초 연습문제  →  [3단계] 실전 문제
```

각 단계는 이전 단계를 완료하지 않으면 **잠금 상태**로 다음 단계 진입이 불가하다. 스킵 없음.

### 1-2. 기술 스택

| 항목 | 선택 | 이유 |
|------|------|------|
| 마크업/스타일 | HTML5 + CSS3 | 브라우저 직접 실행, 별도 앱 설치 불필요 |
| 스크립트 | Vanilla JavaScript (ES6+) | 외부 의존성 최소화, 빠른 로드 |
| 수식 렌더링 | **KaTeX** (CDN) | MathJax보다 훨씬 빠름, 오프라인 fallback 가능 |
| 애니메이션 | CSS Animation + JS | 외부 라이브러리 없이 구현 |
| 파일 구조 | 날짜별 HTML 파일 분리 | GitHub 연동 및 누적 관리 용이 |
| 호스팅 | GitHub Pages | 무료, URL 고정, 날짜별 디렉토리 구조 |

### 4-2. 수학 수식 렌더링 (KaTeX)

**이 항목은 수학 단원 구현에서 절대 생략하지 말 것.**

```html
<!-- HTML <head>에 반드시 포함 -->
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/katex@0.16.9/dist/katex.min.css">
<script defer src="https://cdn.jsdelivr.net/npm/katex@0.16.9/dist/katex.min.js"></script>
<script defer src="https://cdn.jsdelivr.net/npm/katex@0.16.9/dist/contrib/auto-render.min.js"
  onload="renderMathInElement(document.body, {
    delimiters: [
      {left: '$$', right: '$$', display: true},   // 블록 수식 (가운데 정렬)
      {left: '$', right: '$', display: false}       // 인라인 수식
    ]
  });"></script>
```

**수식 작성 예시 — 반드시 이 형식으로 작성:**

```
인라인: $a \div 3 \times b = \dfrac{a}{3b}$
블록(별행):
$$
(a+b) \div 3 \times x = \dfrac{a+b}{3x}
$$

분수: \dfrac{분자}{분모}   ← \frac 대신 \dfrac 사용 (더 크게 표시)
곱셈기호: \times
나눗셈: \div
제곱: x^2, a^3
루트: \sqrt{x}
```

**문제 예시 (이미지의 실제 문제를 KaTeX로 구현하는 방법):**

```html
<!-- 보기 ㄱ번: (a÷3)×b = a/3b -->
<div class="quiz-item">
  <span class="item-label">㉠</span>
  <span class="item-expr">$(a \div 3) \times b = \dfrac{a}{3b}$</span>
  <div class="answer-buttons">
    <button onclick="checkAnswer(this, true)">O</button>
    <button onclick="checkAnswer(this, false)">X</button>
  </div>
</div>
```

**객관식 문제 HTML 구조:**

```html
<div class="quiz-card" data-question-id="q1">
  <div class="question-text">
    다음 중 $2x \div \dfrac{3}{4}y$를 기호를 생략하여 나타낸 것은?
  </div>
  <div class="options">
    <button class="option" onclick="selectOption(this, false)">① $\dfrac{2x}{3y}$</button>
    <button class="option" onclick="selectOption(this, false)">② $\dfrac{8x}{3y}$</button>
    <button class="option correct" onclick="selectOption(this, true)">③ $\dfrac{8x}{3y}$</button>
    <button class="option" onclick="selectOption(this, false)">④ $\dfrac{3y}{8x}$</button>
  </div>
  <div class="feedback hidden">
    <!-- 정답/오답 피드백 영역 -->
  </div>
</div>
```

**즉시 채점 로직:**

```javascript
function selectOption(el, isCorrect) {
  const card = el.closest('.quiz-card');
  if (card.dataset.answered) return; // 이미 답한 문제 재클릭 방지
  card.dataset.answered = 'true';

  el.classList.add(isCorrect ? 'correct-selected' : 'wrong-selected');
  
  if (isCorrect) {
    showFeedback(card, '✅ 정답이야! 완전 잘했어~', 'correct');
    triggerConfetti(); // 간단한 축하 파티클
  } else {
    // 정답 버튼 표시
    card.querySelector('.correct').classList.add('show-correct');
    showFeedback(card, '❌ 아쉽! 아래 정답을 확인해봐. 왜 그런지 선생님이 설명해줄게!', 'wrong');
    showExplanation(card); // 오답 설명 표시
  }
}
```

## 5. 과목별 구현 특이사항

### 5-1. 수학 (최우선 구현 대상)

- **KaTeX 필수** (분수, 제곱, 루트, 비례식, 등호, 부등호 모두 KaTeX로)
- 풀이 과정 step-by-step 단계별 공개
- 수직선, 좌표계 등은 SVG로 직접 구현
- 계산 연습은 input 박스 + 정답 비교 방식 (앞뒤 공백 trim 처리)
- **1학기 목차 기준 구현 순서:**
  1. 소인수분해
  2. 정수와 유리수
  3. **문자와 식** ← 1순위 구현 (첨부 이미지 문제 유형)
  4. 일차방정식
  5. 좌표와 그래프

### 5-2. 국어

- 지문 읽기 → 문제 풀기 구조
- 지문 내 중요 어구 **hover 시 설명 팝업**
- 어휘 카드 (단어 → 클릭 → 뜻 공개, 플래시카드 방식)

### 5-3. 영어

- 단어 학습: 플래시카드 (한→영, 영→한 양방향)
- 듣기 대신 **Web Speech API**로 발음 들려주기 (TTS, 무료)
- 문법 설명 시 한국어/영어 병기 (대조 방식)

### 5-4. 과학

- 개념도(Concept Map)는 SVG 또는 CSS Flexbox로 구현
- 실험 과정은 단계별 일러스트 (CSS로 간단한 도형 조합)
- 용어 정리 카드 (과학 용어 → 정의 → 실생활 예시)

### 5-5. 사회

- 지도/연표는 SVG로 구현 (이미지 외부 의존 없이)
- 역사 단원: 타임라인 컴포넌트
- 핵심 개념: 마인드맵 스타일 카드 배치


해당 파일 하나에 아래를 모두 포함한다:

1. `<head>` — KaTeX CDN 로드, CSS 인라인 (외부 파일 없이 단독 실행 가능하게)
2. **상단 진행바** — 1강의 / 2기초문제 / 3실전문제 (현재 단계 강조)
3. **1단계: 강의 섹션**
   - 도입: "우리가 수학에서 곱셈이랑 나눗셈 기호를 왜 생략하는지 알아?" 식의 대화체 도입
   - 핵심 규칙 4가지 카드 (×생략, ÷→분수, 계수 앞, 1 생략)
   - 위 이미지의 문제 ㉠~㉥번 유형 각각 step-by-step 풀이 예제
   - "강의 다 들었어요!" 버튼
4. **2단계: 기초 연습문제** (7문제, 잠금 상태→강의 완료 후 해제)
   - 위 이미지의 문제 유형과 동일한 형식으로 출제
   - KaTeX 수식 포함, 즉시 채점
5. **3단계: 실전 문제** (5문제, 잠금 상태→기초 70점 이상 후 해제)
6. **완료 화면** — 격려 메시지, 오늘 점수, "내일 또 만나요!" 애니메이션

---

## 9. 기타 구현 주의사항

- **외부 이미지 URL 절대 사용 금지** — CDN 이미지가 언제 사라질지 모름. 모든 시각자료는 SVG, CSS, 또는 Base64 inline으로
- **오디오** — 외부 mp3 파일 없이 Web Audio API로 효과음 생성 (정답음, 오답음, 완료음)
- **인터넷 없이도 동작** — CDN은 KaTeX와 Google Fonts만 허용, 나머지는 완전 로컬
- **반응형** — 스마트폰에서도 수식이 깨지지 않게 (`overflow-x: auto` on math blocks)
- **저장 자동화** — 페이지를 닫았다 다시 열어도 진행 상태 유지 (localStorage)
- **글씨 크기** — 기본 18px (중학생 눈에 피로하지 않게)

## 10. 품질 기준 (완성 판단 기준)

완성된 파일은 아래 체크리스트를 모두 통과해야 한다:

- [ ] KaTeX 수식이 브라우저에서 올바르게 렌더링됨
- [ ] 강의 읽기 전에는 2단계 버튼이 비활성화(disabled/회색) 상태임
- [ ] 기초문제 70점 미만 시 3단계 잠금 유지 + 재도전 메시지 표시
- [ ] 오답 시 즉시 오답 설명이 나타남 (팝업 or 인라인)
- [ ] step-by-step 풀이에서 "다음" 클릭 시 단계적으로 공개됨
- [ ] 모바일(360px 너비)에서 수식이 가로 스크롤 없이 표시됨
- [ ] 페이지 새로고침 후에도 진행 상태 유지됨
- [ ] 과외 선생님 말투로 일관되게 작성됨 (교과서식 문체 없음)
