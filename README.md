# 📚 중학교 1학년 수학 스마트 학습서 (AI 과외선생님)

## 🤖 Antigravity 인수인계서 (Handover Document)

안녕! 나는 이 프로젝트를 처음 세팅한 Antigravity야. 
원장님(User)이 회사 노트북이나 다른 PC에서 너를 다시 부르면, 이 문서를 읽고 바로 다음 작업을 이어서 할 수 있도록 프로젝트 현황과 규칙을 정리해둘게.

### 1. 프로젝트 개요
*   **목표:** 중학교 1학년 아들의 수학 학습을 위한 "스마트 학습서" 구축
*   **주요 대상 단원:** 문자와 식 (가장 중요)
*   **개발 환경:** 오로지 노트북 PC (와이드 화면에 최적화된 Desktop UI)
*   **특징:**
    *   3분할 레이아웃 (좌측 내비게이션, 중앙 학습 화면, 우측 AI 챗봇)
    *   `gemini-3.1-flash-lite` 모델이 연동된 우측 챗봇 패널 (정답을 알려주지 않고 힌트만 제공하는 과외 선생님)
    *   객관식 3 : 주관식 7 비율의 풍부한 문제 (하루 기초 15문제 + 실전 15문제 = 총 30문제)
    *   `localStorage`를 활용한 진도 및 점수 저장, 그리고 다운로드(Export)/가져오기(Import) 기능으로 기기 간 데이터 연동

### 2. 폴더 구조
*   `index.html`: 메인 대시보드 (진행률 확인 및 데이터 Import/Export)
*   `days/`: 일차별 학습 파일이 저장되는 폴더
    *   `2026-06-12/math_char_expr_1.html` (1일차: 곱셈, 나눗셈 기호 생략)
    *   `2026-06-13/math_char_expr_2.html` (2일차: 대입과 식의 값)
    *   `2026-06-14/math_char_expr_3.html` (3일차: 일차식의 계산)
*   `builder/` (선택): 초기에 HTML을 찍어내기 위해 만든 스크립트가 있지만, 현재는 각 HTML 파일이 독립적으로 완벽하게 동작하므로 직접 파일을 복사해서 수정하는 방식을 권장해.

### 3. 다음 작업(내일 자 학습지) 준비 가이드
네가 이 저장소를 클론(Clone)하고 새로운 일차(예: 4일차)를 만들어야 한다면 다음 순서로 진행해줘.

1.  `days/` 폴더 내에 새로운 날짜 폴더(예: `2026-06-15/`)를 생성해.
2.  가장 최근 HTML 파일(예: `math_char_expr_3.html`)을 복사해서 새 폴더에 넣어 (예: `math_char_expr_4.html`).
3.  새 파일에서 다음 내용을 수정해:
    *   `<title>`, `<h1>` 제목 업데이트 (예: 4일차: 일차방정식의 풀이)
    *   `dateKey` 와 `unitKey` 변수 업데이트 (예: `const dateKey = "2026.06.15 (월)"; const unitKey = "math_4";`)
    *   `stage-done` 부분의 "다음 진도 나가기" 링크 업데이트
    *   **문제 교체:** 기초 15문제, 실전 15문제를 새 단원 내용에 맞게 생성 (주관식 `checkSubjective`, 객관식 `checkAnswer` 로직은 기존 코드를 재사용)
4.  `index.html` 의 메인 대시보드에 4일차 목록을 추가하고, `checkStatus` 함수 호출 개수를 늘려서 전체 미션 완료 카운트 로직을 수정해줘.

### 4. 주의사항 (원장님 특별 요청)
*   **절대 정답 스포일러 금지:** 챗봇 프롬프트를 수정할 일이 있다면, 챗봇이 정답을 직접 말하지 않게 하는 룰을 꼭 유지해줘.
*   **주관식 채점:** 주관식은 띄어쓰기나 대소문자를 무시하도록 `normalizeString()` 함수가 적용되어 있어. 정답 배열(`['5x', '5*x']` 등)에 예상 답안을 다양하게 넣어줘.
